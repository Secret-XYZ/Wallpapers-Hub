# Wallpapers-Hub

Wallpapers-Hub - инструмент для быстрого переключения обоев и просмотра их в 1 приложении.

# Использование

Запустите .bat файл приложения, после у вас автоматикески откроеться меню с выбором папки с фото. Можно выбрать папку с обоями которая уже идет в проекте, чтобы начать - выберите ее или свою папку с фото у себя на компьютере или введите путь своей папки. После выберите справва меню желаемые обои, нажмите на их и нажмите кнопку установвить обои. В папку так же можно загружать любые фотографии в формате .png.
